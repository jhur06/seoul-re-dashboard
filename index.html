주간 데이터는 KB부동산과 한국부동산원의 주간 가격동향조사 기반입니다. 매주 목요일 공표되며, PublicDataReader를 통해 API 키 없이 수집합니다. 외국인 거래 데이터는 주간 단위로 제공되지 않으며, 월별(IROS) 또는 반기별(KOSIS)로만 확인 가능합니다.
